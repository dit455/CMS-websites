# CMS → Website Section Guide

Quick reference: which CMS admin section controls which part of the website.

---

## HEADER

| What you see on the website | Where to change in CMS Admin |
|---|---|
| Department name (big title) | **Portal Settings → Site Settings** → Department Name |
| Government name (below dept) | **Portal Settings → Site Settings** → Government Name |
| Nav menu links | **Menu Manager → Menu Items** |
| Contact email in top bar | **Portal Settings → Site Settings** → Helpdesk Email |

---

## NEWS TICKER (scrolling bar at the top)

| Website | CMS Admin |
|---|---|
| Scrolling news text | **News → News Ticker Items** |

---

## HERO SLIDER (big banner with image)

| Website | CMS Admin |
|---|---|
| Slide image, title, description | **Hero Banner → Hero Banner Slides** |
| Stats bar below slider (e.g. "24x7 Digital access") | **Portal Settings → Hero Stats** |

---

## DEPARTMENT OVERVIEW

| Website | CMS Admin |
|---|---|
| Department name (heading) | **Portal Settings → Site Settings** → Department Name |
| Description paragraph | **Portal Settings → Site Settings** → Footer Description |
| Department / Government / Helpdesk info | **Portal Settings → Site Settings** |
| Quick access cards grid (right side) | *(add via code in `extensions/<site-key>/index.jsx` or `defaultContent.js`)* |

---

## OFFICIALS (Leadership & Officials)

| Website | CMS Admin |

## SERVICES

| Website | CMS Admin |

## ACTIVITIES

| Website | CMS Admin |

## CITIZEN RESOURCES (Quick Links + Resource Cards)

## FOOTER

| Website | CMS Admin |
|---|---|
| Department name | **Portal Settings → Site Settings** → Department Name |
| Government name | **Portal Settings → Site Settings** → Government Name |

## ABOUT PAGE (opens when clicking "Detailed About Us")

| Website | CMS Admin |
|---|---|

## SUMMARY — Most used sections

| Priority | CMS Section | Controls |
|---|---|---|
| 1 | **Portal Settings → Site Settings** | Header title, footer, contact info |
| 2 | **Menu Manager → Menu Items** | Navigation links |
| 3 | **Hero Banner → Hero Banner Slides** | Homepage banner |
| 4 | **Portal Settings → Leadership & Officials** | Officials section |
| 5 | **Services → Services** | Services cards |
| 6 | **Notifications → Notifications** | Notifications section |
| 7 | **Documents → Documents** | Documents section |
| 8 | **News → News Ticker Items** | Scrolling news bar |
| 9 | **Portal Settings → Connected Platforms** | Partner logos |
| 10 | **Portal Settings → Hero Stats** | Stats bar on hero |
